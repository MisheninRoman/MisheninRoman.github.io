Калькулятор
Данный калькулятор был реализован с помощью CSS свойств display: flex и display: grid.

Использование display:flex позволило легко выстроить все элементы в нужном порядке и задать им необходимые пропорции.

Это помогло создать гибкую и адаптивную верстку, которая отлично смотрится на разных устройствах.

Использование display: grid позволило разбить калькулятор на ячейки и удобно расположить кнопки в нужных местах.

Это позволило легко настраивать внешний вид калькулятора и управлять его поведением.

В общем, использование этих CSS свойств помогло создать простой и эффективный калькулятор, который легко может быть доработан и модифицирован по желанию.

Крестики-нолики
Данная игра крестики-нолики была реализована с помощью CSS свойства display: flex и HTML тегов table/tr/td.

Использование display: flex позволило легко выстроить все элементы в нужном порядке и задать им необходимые пропорции.

HTML теги table/tr/td были использованы для создания сетки, на которой отображаются крестики и нолики.

Благодаря этому была создана простая и эффективная игра, которая легко может быть доработана и модифицирована по желанию.

Кроме того, использование HTML тегов позволило создать более семантичный и структурированный код.
